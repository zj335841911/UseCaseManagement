# 数据字典-配置范围（ConfigScope）
## 字典说明
配置范围

## 基本说明
| 项目 | 说明 |
| ---- | ---- |
| 数值类型 | 字符串 |
| 启用多选 | 否 |
| 多选分隔符 | 无 |
| 启用缓存 | 是 |
| 缓存超时时长 | 无 |

## 取值范围
| 序号 | 数值 | 文本 | 父值 | 提示信息 | 是否显示空白 | 默认选择 | 说明 |
| ---- | ---- | ---- | ---- | ---- | ---- | ---- | ---- |
| 1 | sys | 全局 |  |  | 否 | 否 | 全局 |
| 2 | org | 当前组织 |  |  | 否 | 否 | 当前组织 |
| 3 | dept1 | 当前部门（含子部门） |  |  | 否 | 否 | 当前部门（含子部门） |
| 4 | dept2 | 当前部门（不含子部门） |  |  | 否 | 否 | 当前部门（不含子部门） |
| 5 | user | 个人 |  |  | 否 | 否 | 个人 |

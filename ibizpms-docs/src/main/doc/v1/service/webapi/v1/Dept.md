## 附录
### 数据类型说明
#### DeptDTO
| 序号 | 属性名 | 属性类型 | 是否可以为空 | 说明 |
| ---- | ---- | ---- | ---- | ---- |
| 1 | manager | String | 允许 | 负责人 |
| 2 | isleaf | String | 允许 | 无子部门 |
| 3 | grade | Integer | 允许 | grade |
| 4 | function | String | 允许 | function |
| 5 | order | Integer | 允许 | order |
| 6 | path | String | 允许 | path |
| 7 | position | String | 允许 | position |
| 8 | id | Long | 不可 | id |
| 9 | name | String | 不可 | 部门名称 |
| 10 | parentname | String | 允许 | 上级部门 |
| 11 | parent | Long | 允许 | parent |
| 12 | <动态属性> | Object | 允许 | 支持动态属性 |

#### DeptSearchContext
| 序号 | 属性名 | 属性类型 | 是否可以为空 | 说明 |
| ---- | ---- | ---- | ---- | ---- |
| 1 | n_name_like | String | 允许 | 条件字段：name<br>条件组合方式：`%like%` |
| 2 | n_parentname_eq | String | 允许 | 条件字段：parentname<br>条件组合方式：`=` |
| 3 | n_parentname_like | String | 允许 | 条件字段：parentname<br>条件组合方式：`%like%` |
| 4 | n_parent_eq | Long | 允许 | 条件字段：parent<br>条件组合方式：`=` |
| 5 | customcond | String | 允许 | 自定义查询条件 |
| 6 | customparams | String | 允许 | 自定义查询参数 |
| 7 | query | String | 允许 | 快速搜索 |
| 8 | filter | QueryFilter | 允许 | 条件表达式<br>参照`cn.ibizlab.pms.util.filter.QueryFilter` |
| 9 | page | int | 允许 | 当前页数<br>默认值0 |
| 10 | size | int | 允许 | 每页显示条数<br>默认值20 |
| 11 | sort | String | 允许 | 排序 |
